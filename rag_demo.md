现在要制作一个RAG demo，演示如何使用RAG来回答用户的问题。
要实现的功能点
1. 用户输入问题，点击搜索查询问题
2. 系统根据问题从知识库中提取相关内容
3. 系统将提取到的内容与问题一起输入到LLM中
4. LLM根据问题和相关内容生成回答
5. 系统将回答返回给用户
6. markdown文档上传功能，用户可以上传自己的文档到知识库中，系统会自动提取文档内容并存储到知识库中，下次查询时可以基于该文档内容回答问题。
技术栈：
后端：
python(uv管理) +langgraph+阿里云的embedding模型+RAG+deepseek llm
前端：
Next.js+Tailwind CSS
注意：
仅实现基本功能，不考虑性能优化和安全问题。配置环境变量文件，包括阿里云的embedding模型的访问密钥等，暂时先mock,deepseek llm的访问密钥也需要配置到环境变量中。
